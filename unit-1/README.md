#### Terminal, CLI, Git, & GitHub

- Version Control

- Data Types, Variables, Primitives
- Arrays

- Basic and Advanced Functions
- Objects
- Callbacks
- Testing

- CSS basic styles and layouts (Grid & Flexbox)

- Events
- Servers & Hosting

#### Computer Science

- Recursive Functions
- Basic Sorting Algorithms
